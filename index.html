About

My name is Gerry Straatsma, I grew up and still live in Brampton Ontario. I have always had an interest in computers and technology. That interest only grew once I started to learn how code works when I was 14. I started by learning HTML and CSS. Creating my first websites helped me by teaching me the basics of debugging code and how to research about a language. Later I was introduced to Java in my first computer science class. When I was starting to learn Java, my interest grew into a passion. I wanted to learn and see what I was capable of and push my limits. I have always like pushing myself by working on small or large projects that push me to learn more in order to get to the end of the project. Starting a project while knowing I have no idea how one aspect is supposed to work is my favourite thing because it will force me to expand my knowledge.
